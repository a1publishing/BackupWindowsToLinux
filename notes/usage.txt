2. Install to your module library (but also see update build commands below);
  The build produces a versioned folder bin\BackupWindowsToLinux\2.1.0\. Your module library uses the same versioned structure, so just copy
   that folder:

Copy-Item -Path "H:\dev\20260213_claude_backupToLinux\bin\BackupWindowsToLinux\2.1.0" `
            -Destination "S:\lib\pow\mod\BackupWindowsToLinux\2.1.0" `
            -Recurse

  Then test with:
Remove-Module BackupWindowsToLinux -Force -ErrorAction SilentlyContinue
Import-Module BackupWindowsToLinux -RequiredVersion 2.1.0 -Force

  You don't need to touch or overwrite the old 2.0.0 folder — PowerShell will load the highest version by default, so 2.1.0 will take over
  automatically.
